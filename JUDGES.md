## Try the agent API yourself (no login)

The killer narrative — "AI agents pay AI agents" — runs without any signup. The reference implementation is a self-contained ~200-line Python script:

```bash
git clone https://github.com/balflee/GapSmith
cd GapSmith
pip install solders solana spl-token requests base58
python examples/agent_demo.py --skip-compute
```

It generates an ephemeral devnet wallet (or accepts your own with `--secret-key`), hits the production API, gets HTTP 402, signs a USDC `transferChecked`, retries with `X-PAYMENT`, and unpacks the data. Total spend: ~$0.55 on devnet (free, faucet-funded).

For mainnet (real USDC):

```bash
python examples/agent_demo.py --mainnet --secret-key <YOUR_BASE58>
```

API docs and the full endpoint reference: https://gapsmith.draftlabs.org/docs/api
OpenAPI spec: https://gapsmith.draftlabs.org/api/v1/openapi

## How to verify everything is real

- **On-chain**: every USDC payment lands at the merchant address linked above. Click any tx in your Phantom history to see it on Solscan.
- **API auditability**: the `agent_jobs` table records every paid x402 call with `tx_hash UNIQUE` for idempotency — replay attacks return the cached response (sync) or 409 (async).
- **Code**: this repo is the full live product. Everything you see at `gapsmith.draftlabs.org` is built from `main`.
